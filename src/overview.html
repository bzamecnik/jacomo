JaCoMo (Jabber Contact Monitor) is a an application to monitor presence of Jabber contacts and show it in a graphical way along with some statistics. It consists of two parts:

  • Jabber bot which acts as a Jabber client. It collects changes in presence of contacts in roster and stores them into a database.
  • Data interpreter which takes the presence data stored in the database, computes some statistics and show it in a graphical way.
